# 🚀 Live Demo

The full dashboard is deployed and publicly accessible — no setup required.

## 🔗 [jetrul-app.streamlit.app](https://jetrul-app.streamlit.app/)

The app lets you:
- Select any test engine and see its predicted RUL trajectory across all three models
- Compare Random Forest, XGBoost, and CNN-LSTM predictions side by side
- Explore EDA plots and sensor trends interactively
- View cluster health states from the K-Means analysis
